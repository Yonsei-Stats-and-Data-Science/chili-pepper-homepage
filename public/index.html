Chili Pepper

HPC Data Science Cluster


Chili Pepper is a High Performance Computing(HPC) cluster configured to run heavy data science workloads via SLURM. Administrators are currently working on a web-based Jupyter sessions. Currently 2 compute nodes are available. The CPU dedicated server is equipped with 32 physical cores and 128GB of RAM. The GPU node has 16 physical cores and 80GB of RAM with 2 Tesla T4 GPUs with 16GB of vRAM each. The cluster is hosted via Naver Cloud Platform and configured by student administrators. This project is funded by BK21 Foundation and Yonsei University Department of Statistics and Data Science.
To gain access to this server please send an email to the administrator.
Thank You.
